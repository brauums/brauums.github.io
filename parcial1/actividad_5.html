Siglas

LAN
Local Area Network (Red de área local): una red de computadoras que cubre un área geográfica pequeña, como una oficina o un edificio.
IP
Internet Protocol (Protocolo de Internet): protocolo de comunicación utilizado para la identificación y localización de dispositivos en una red.
TCP
Transmission Control Protocol (Protocolo de control de transmisión): protocolo de comunicación que garantiza la entrega confiable de datos en una red.
UDP
User Datagram Protocol (Protocolo de datagramas de usuario): protocolo de comunicación que permite la transmisión rápida de datos sin garantizar su entrega.
DNS
Domain Name System (Sistema de nombres de dominio): sistema que traduce nombres de dominio en direcciones IP.
HTTP
Hypertext Transfer Protocol (Protocolo de transferencia de hipertexto): protocolo utilizado para la transferencia de información en la web.
HTTPS
Hypertext Transfer Protocol Secure (Protocolo de transferencia de hipertexto seguro): versión segura de HTTP que usa cifrado SSL/TLS.
MAC
Media Access Control (Control de acceso al medio): dirección única asignada a una tarjeta de red para identificarla en una red.
VPN
Virtual Private Network (Red privada virtual): tecnología que permite una conexión segura y cifrada a través de una red pública.
DHCP
Dynamic Host Configuration Protocol (Protocolo de configuración dinámica de host): protocolo que asigna direcciones IP automáticamente a dispositivos en una red.
FTP
File Transfer Protocol (Protocolo de transferencia de archivos): protocolo utilizado para transferir archivos entre dispositivos en una red.
